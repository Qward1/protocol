# Frontend — Цифровой Офис

React + Vite + TypeScript + Tailwind. Светлая/тёмная темы, загрузка медиа,
кликабельный транскрипт с аудиоплеером, протоколы и поручения, справки-обоснования,
Q&A с выбором контекста, семантический поиск, экспорт, дашборд контроля исполнения.

## Запуск

```bash
cd frontend
npm install
npm run dev          # http://localhost:5173 (проксирует /api -> :8000)
```

Backend должен быть запущен на `http://localhost:8000` (см. `../backend/README.md`).
Прокси настроен в `vite.config.ts`.

## Структура

```
src/
  lib/        api.ts (клиент REST + типы), theme.tsx (light/dark), utils.ts
  store/      selection.ts (zustand: scope Q&A + команда seek плеера)
  components/ Layout, ui, ExportMenu, JustificationModal
  pages/      Dashboard, Upload, Transcript, ProtocolsList, Protocol, Library, Chat
```

## Темизация

Палитра — CSS-переменные в `src/index.css`, переключаются классом `dark` на `<html>`.
Тема сохраняется в localStorage и применяется до рендера (скрипт в `index.html`).

## Основные сценарии

1. **Загрузка** → drag-drop файла → переход на страницу транскрипта (поллинг статуса).
2. **Транскрипт** → аудиоплеер + кликабельные сегменты (клик = перемотка), диаризация цветом.
3. **Протокол** → текст + таблица поручений; кнопка «Обоснование» открывает справку (фрагмент + должностная обязанность).
4. **Библиотека** → выбор записей/протоколов как контекста + семантический поиск.
5. **Вопросы** → чат по выбранному контексту с цитатами-ссылками.
6. **Экспорт** → DOCX/PDF/MD/TXT/JSON на страницах транскрипта, протокола, справки, чата.
